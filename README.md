# Social-Tree


This is a tool that allows you to share multiple social media links
## Run Locally

Clone the project

```bash
  git clone git@github.com:vicodevv/social-tree.git
```

Go to the project directory

```bash
  cd social-tree
```

Install dependencies

```bash
  npm install
```

Run the code

```bash
  npm start
```
